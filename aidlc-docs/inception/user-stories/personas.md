# User Personas (프리미엄 카페)

## Persona 1: 고객 (Customer)

- **이름**: 김고객
- **역할**: 프리미엄 카페 방문 고객
- **목표**: 나만의 커스텀 커피를 편리하게 주문하고 싶다
- **특성**:
  - 테이블에 비치된 태블릿을 통해 주문
  - 별도 로그인/회원가입 없이 즉시 사용
  - 커피 옵션(사이즈, 온도, 샷, 시럽, 우유 종류 등)을 직접 선택
- **Pain Points**:
  - 직원에게 복잡한 커스텀 옵션을 구두로 전달하기 어려움
  - 주문 상태를 알 수 없는 불안감
- **Success Criteria**: 커스텀 옵션 선택부터 주문 완료까지 직관적으로 가능

---

## Persona 2: 매장 관리자 (Store Admin)

- **이름**: 이매니저
- **역할**: 카페 매장 운영 담당자
- **목표**: 실시간으로 커스텀 주문을 관리하고 매장을 효율적으로 운영하고 싶다
- **특성**:
  - 매장 내 주문/테이블/메뉴 및 옵션을 관리
  - 커스텀 옵션이 포함된 주문 상세 확인 필요
- **Pain Points**:
  - 복잡한 커스텀 주문 누락 또는 오류
  - 메뉴별 옵션 관리 번거로움
- **Success Criteria**: 커스텀 옵션 포함 주문을 정확하게 확인하고 처리

---

## Persona 3: 슈퍼 관리자 (Super Admin)

- **이름**: 박총괄
- **역할**: 전국 카페 체인 총괄 관리자
- **목표**: 전국 카페 매장의 주문 현황을 통합 모니터링하고 시스템을 관리하고 싶다
- **특성**:
  - 전국 매장 생성/관리, 매장 관리자 계정 관리
  - 전국 매장 주문 실시간 모니터링
  - 시스템 설정 관리
- **Pain Points**:
  - 전국 매장별 주문 상황 파악 어려움
  - 매장/계정 관리 번거로움
- **Success Criteria**: 전국 모든 매장의 주문 현황을 한 화면에서 실시간 확인 가능
